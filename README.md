🛒 Mini Shopping Mart

A responsive multi-page e-commerce website built using HTML, CSS, and JavaScript.
This project simulates an online shopping experience with product browsing, cart functionality, and basic user authentication.
A responsive multi-page e-commerce website built using HTML, CSS, and JavaScript. This project is designed to simulate a real-world online shopping experience, allowing users to browse products, view detailed product information, manage a shopping cart, and interact with a simple authentication system. The focus of this project is on creating a smooth user experience with clean design, intuitive navigation, and interactive frontend features without relying on any backend technologies.

The website includes a structured multi-page layout consisting of a Home page, multiple category pages such as Groceries, Clothing, Personal Care, and Home, as well as a dedicated Product Details page. Each product is linked to a detailed view where users can see pricing, description, and ratings before adding items to the cart. The cart system is fully functional and allows users to add products, adjust quantities using increase/decrease buttons, and view total pricing dynamically.

To enhance usability, the project uses LocalStorage to store cart data and user login information, ensuring that user actions persist even after refreshing the page. A basic Login and Signup system is implemented to simulate user authentication, providing restricted access to the shopping interface. Additionally, a search feature is included to help users quickly find products, improving navigation and accessibility.

The interface is designed to be responsive, ensuring compatibility across different screen sizes including mobile, tablet, and desktop devices. Modern UI elements such as toast notifications are used to provide feedback for actions like adding items to the cart, making the interaction more engaging and user-friendly.

This project demonstrates strong fundamentals in frontend development, including DOM manipulation, event handling, local data storage, and responsive design principles. It serves as a solid foundation for building more advanced full-stack e-commerce applications in the future.
