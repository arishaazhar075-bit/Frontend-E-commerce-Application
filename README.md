🛒 Mini Shopping Mart

A responsive multi-page e-commerce website built using HTML, CSS, and JavaScript.
This project simulates an online shopping experience with product browsing, cart functionality, and basic user authentication.

🚀 Features
🏠 Multi-page layout (Home, Categories, Product Details)
🛍️ Add to Cart functionality
🔢 Quantity management (increase/decrease items)
💾 Cart data stored using LocalStorage
🔐 Basic Login & Signup system
🔍 Product search functionality
🔔 Toast notifications for better user experience
📱 Responsive design for different screen sizes
🗂️ Pages Included
Home Page
Categories (Groceries, Clothing, Personal Care, Home)
Product Details Page
Cart Page
Login & Signup Pages
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage
